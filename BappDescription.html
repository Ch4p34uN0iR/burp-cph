CPH provides a simple way to modify ANY part of an HTTP message, thus allowing manipulation of these parts with surgical precision even (and especially) when using macros. Typically, macros only support HTTP parameters (name=value), but with this extension anything can be targeted as if it were a parameter!

    Features include:
  • Automatic modification of Requests, Responses, or Both based on scope and pattern matching
  • Find and Replace or Find and Insert of values, with support for RegEx
  • Burp Suite Tools Scoping, only modify traffic from specific areas of Burp
  • Matching specific indices and/or slices
  • Chaining and Ordering of modifications, and the ability to base next-in-line modifications off previous tab responses
    • Save/Load Config:
    • Quicksave/Quickload (persists between Burp/Extender sessions)
    • Import/Export from and to file
  • Extracting replace values from final macro responses
  • Caching messages for full macro modification

More information and bug reporting on github.com/elespike.burp-cph

Requires Jython version 2.7.0
